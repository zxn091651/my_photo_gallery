# zxn's Photo Gallery

一个用于浏览移动硬盘照片和视频的个人相册网页。

## 功能

- 检测后端、隧道、后端服务和移动硬盘的连接状态。
- 通过移动硬盘序列号识别硬盘，不依赖固定盘符。
- 自动定位移动硬盘里的 `影像备份` 文件夹。
- 在网页首页分页展示 `影像备份` 下的子文件夹。
- 点击文件夹后，以卡片滑动方式查看照片和视频。
- 支持随机探索所有子文件夹里的媒体文件。
- 支持照片、视频、HEIC、HEIF、TIFF、AVIF 等常见格式预览。
- 视频预览使用首帧缩略图。
- 查看时可下载当前照片或视频。
- 查看时可输入序号跳转到第 N 张。
- 上传照片或视频到已有子文件夹，或新建一个同级子文件夹后上传。
- 上传需要密码验证，读取和下载不需要密码。
- 不提供任何删除功能。
- 后端控制器可隐藏到系统托盘，双击托盘图标可重新打开。
- 隧道异常时，控制器会尝试自动重启隧道。

## 使用前准备

1. 插入移动硬盘。
2. 确认移动硬盘中存在 `影像备份` 文件夹。
3. 确认 `.env` 已配置：

```env
PORT=8787
HOST=0.0.0.0
GALLERY_MEDIA_FOLDER=影像备份
GALLERY_DISK_SERIAL=你的移动硬盘序列号
GALLERY_UPLOAD_PASSWORD=你的上传密码
CHMLFRP_FRPC_PATH=frpc.exe 的路径
CHMLFRP_CONFIG_PATH=隧道 ini 配置路径
GALLERY_PUBLIC_STATUS_URL=公网 /api/status 地址
```

## 启动后端

双击运行：

```text
gallery-backend-control.exe
```

打开后：

1. 在隧道配置中选择对应隧道，例如 `zxn_photo_gallery`。
2. 点击 `启动后端`。
3. 等待链路状态变为绿色。
4. 打开前端网页访问相册。

## 关闭或隐藏控制器

- 点击窗口右上角 `X`：控制器不会退出，只会隐藏到系统托盘。
- 双击托盘图标：重新显示控制器窗口。
- 右键托盘图标，选择 `关闭控制器`：真正退出控制器。

## 停止后端

打开控制器后点击：

```text
停止后端
```

这会停止本机后端和当前选择的隧道。

## 上传文件

1. 在网页首页点击 `上传`。
2. 选择目标文件夹，或输入新建文件夹名称。
3. 选择照片或视频文件。
4. 输入上传密码。
5. 点击上传并等待进度完成。

上传成功后，文件会保存到移动硬盘的 `影像备份` 对应文件夹中。

## 常用命令

安装依赖：

```bash
npm install
```

本地启动后端：

```bash
npm run start
```

重新生成控制器：

```bash
npm run launchers:build
```
